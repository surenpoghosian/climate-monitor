# AUA Summer Engineering School
### Climate Monitor

This Arduino project utilizes a number of digital and analog sensors as well as radio modules.
It lets you to observe, transfer and visualize the properties below...

- Temperature
- Humidity
- Atmosphere Pressure
- Wind Direction
